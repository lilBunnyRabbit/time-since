# Time Since

A progressive web app for tracking time since events happened. Create timers, reset them, and view the history of resets with duration between each one.

## Features

- Create named timers to track time since an event
- Reset timers and automatically log the reset history
- View time between each reset
- PWA - installable and works offline
- Data persisted in browser storage

## Quick Start

```bash
bun install
bun dev
```

## Build

```bash
bun run build
bun run preview
```

## Tech Stack

- **Svelte 5** - UI framework with runes
- **TypeScript** - Type safety
- **Vite** - Build tool and dev server
- **Tailwind CSS** - Styling
- **vite-plugin-pwa** - PWA support
